#  Tenzies Game

A fun and interactive **Tenzies dice game** built using **React**, **Tailwind CSS**, and **Vite**.  
Roll the dice until all dice show the same number — but you can freeze dice between rolls to strategize your win!

---

##  Features

-  Roll 10 dice with random values (1–6)
-  Click a die to **hold/freeze** its value
-  Re-roll only the **unheld dice**
-  Win condition detection (all dice held + same value)
-  Confetti celebration on winning
-  Start a new game after winning
-  Responsive UI using Tailwind CSS

---

##  Tech Stack

- **React** (Hooks: `useState`, `useEffect`)
- **Vite** (fast development & build tool)
- **Tailwind CSS** (styling)
- **react-confetti** (win animation)
